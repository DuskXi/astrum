# Internal API Reference

Internal APIs are primarily for Astrum's implementation, tests, debugging, and advanced integrations. They may change between versions, so regular application code should avoid depending on them.

## Index

- [Scheduler Internals](scheduler.md): helper methods on `DynamicScheduler`.
- [Planning Internals](planner.md): `ExecutionPlanner` and execution plan models.
- [Data Transport Internals](data-transport.md): generation, completion, validation, and visualization helpers.
- [Type Tools](type-tools.md): advanced type matching helpers.
